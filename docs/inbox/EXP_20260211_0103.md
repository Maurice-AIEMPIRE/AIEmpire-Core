# Experiment Design for Testing Hypothesis "H5"

## HYPOTHESIS:
"DMs with 'Value First' get three times more replies."

## EVIDENCE REQUIRED:

1. Number of DMs sent using the standard format (control group) and number containing 'Value First'.
2. Total responses received from each DM type.
3. Distribution breakdown: positive, negative or neutral feedback for both groups.

## FASTEST EXPERIMENT:
To get 80% confidence in this hypothesis we can perform an A/B test with at least two weeks of data collection to reach statistical significance and ensure a diverse sample pool representing different demographics (gender, age group etc).

The key metric is "Total Responses" divided by 2 for both groups (standard DM and ValueFirst). In other words, total number of replies per day/duration. We are interested in the difference between them which should be three times greater if our hypothesis holds true.

## DECISION RULE:
If "Total Responses" for DMs containing 'Value First' is 3x larger compared to control group, then we conclude that H5 has been proven. If not (i.e., less than two times the response rate), it can be rejected.
The statistical significance should reach at least a confidence level of 80%.
