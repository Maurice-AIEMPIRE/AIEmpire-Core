# Templates

This directory contains reusable templates for various purposes.

## Structure

```
templates/
├── outreach/         # Email & DM templates
├── packages/         # Customer package templates
├── content/          # Content templates
└── documents/        # Document templates
```

## Template Categories

### Outreach Templates
- Cold email sequences
- DM templates
- Follow-up messages
- Meeting requests

### Package Templates
- Service offerings
- Pricing sheets
- Proposals
- Contracts

### Content Templates
- Blog post outlines
- Social media posts
- Newsletter formats
- Video scripts

### Document Templates
- Technical documentation
- User guides
- Onboarding docs
- Case studies

## Using Templates

1. Copy template to your working directory
2. Fill in bracketed placeholders: `[Name]`, `[Company]`
3. Customize for specific use case
4. Never edit the original template

## Contributing Templates

1. Create template file with clear structure
2. Use placeholders for variable content
3. Add comments explaining sections
4. Document in this README
5. Submit via pull request

## Template Format

```markdown
# Template Name

## Purpose
What this template is for

## Variables
- [Variable 1]: Description
- [Variable 2]: Description

## Example
Sample filled-in version

## Template Content
---
[Your template content here]
---
```
