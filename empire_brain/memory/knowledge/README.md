# Memory / Knowledge

Extracted and curated knowledge base for the empire's AI systems.

## Purpose
- Store verified facts, decisions, and learnings
- Provide context for AI agent prompts
- Build institutional memory that survives session boundaries

## Categories
- **fixes/** - Bug fixes, workarounds, and technical solutions
- **decisions/** - Business and architecture decisions with rationale
- **patterns/** - Recurring patterns and best practices
- **market/** - Market intelligence, competitor data, trends

## Integration
Used by `antigravity/knowledge_store.py` for persistent knowledge retrieval.
