# Memory / Chats

Storage for imported chat histories from Claude, ChatGPT, Kimi, and other AI sessions.

## Purpose
- Preserve context from important conversations
- Enable cross-session knowledge transfer
- Provide training data for knowledge extraction

## File Naming Convention
```
YYYY-MM-DD_<provider>_<topic>.json
```
Example: `2026-02-19_claude_revenue_strategy.json`

## Supported Formats
- JSON (Claude/ChatGPT exports)
- Markdown (manual conversation logs)
- JSONL (streaming conversation records)
