# Assets

Reusable assets for the empire's operations.

## Purpose
- Store templates, images, and configuration files
- Maintain brand assets and design resources
- Keep reusable content blocks and snippets

## Categories
- **templates/** - Document and email templates
- **images/** - Logos, covers, social media graphics
- **configs/** - Reusable configuration files
- **prompts/** - System prompts and prompt libraries
