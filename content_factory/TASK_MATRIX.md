# Task Matrix (25 Agenten)

## Orchestrator (1)
- Aufgabe: Gesamtziel zerlegen, QA steuern
- Output: Agentenliste + Review-Bericht
- Wortlimit: 120 pro Agent

## Ideation (10)
- Aufgabe: Hooks, Themen, Angles
- Output: 30 Hooks + 20 Themen pro Agent
- Wortlimit: 400

## Writer (8)
- Aufgabe: Drafts fuer Tweets, Threads, Prompts
- Output: formatierte Drafts
- Wortlimit: variabel pro Format

## Refiner (4)
- Aufgabe: kuerzen, schaerfen, Struktur
- Output: optimierte Drafts
- Wortlimit: gleichbleibend

## Strategy (2)
- Aufgabe: Monetarisierung
- Output: Zielgruppe, Offer, Pricing, Funnel, CTAs
- Wortlimit: 600
