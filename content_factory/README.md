# Content Factory (25 Agent System)

This folder contains a complete, ready-to-run content system built around 25 specialized AI agents.
It includes:
- A finished thread in German
- A full agent prompt pack
- A task matrix and workflow runbook
- Output formats and a monetization strategy template

Default assumptions used:
- Niche: AI agents for content creation
- Style: direct, clear, no emojis

If you want a different niche or style, edit the placeholders in the prompt files.
