# AI Agents

This directory contains AI agent configurations, prompts, and policies.

## Structure

```
agents/
├── prompts/          # Agent system prompts
├── roles/            # Agent role definitions
├── policies/         # Behavior policies
└── tools/            # Custom agent tools
```

## Agent Types

### Research Agent
- Scans trends and competitors
- Keyword research
- Market analysis

### Content Agent
- Creates social media posts
- Generates blog articles
- Email sequences

### Sales Agent
- Lead qualification (BANT)
- Outreach automation
- Follow-up sequences

### Community Agent
- Engagement playbooks
- Reply templates
- DM automation

### Analytics Agent
- KPI tracking
- Conversion analysis
- Revenue reporting

## Guidelines

- Use markdown for prompts
- Version control all changes
- Test prompts before deploying
- Document expected outputs

## Prompt Template

```markdown
# Agent Name

## Role
Brief description of agent purpose

## Context
Background information needed

## Instructions
Step-by-step tasks

## Constraints
Limitations and boundaries

## Output Format
Expected response structure
```
