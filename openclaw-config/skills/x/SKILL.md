## Posting-Regeln (9 Posts/Tag)

### Format-Mix pro Tag:
- 3× kurze Insights (1-3 Sätze, kein Thread)
- 2× Threads (5-8 Tweets, Story-Arc)
- 2× Reply-Guy-Posts (auf Trending-Accounts antworten)
- 1× Langform-Tweet (600+ Zeichen, Wert)
- 1× Re-share oder Zitat mit eigenem Kommentar

### Posting-Zeiten (CET):
- 07:30 – Morgendlicher Hook
- 12:00 – Mittags-Insight
- 17:30 – Abend-Thread oder Frage
- 21:00 – Viraler Reply oder Meinung

### Voice-Regeln:
- Kein "Ich freue mich" oder "Schön zu sehen dass"
- Kein Corporate-Speak
- Kurze Sätze. Oft. Wie das.
- Zahlen verwenden (16 Jahre, 3 Systeme, 0 EUR Kosten)
- Deutsch ODER Englisch – nie gemischt in einem Post
- Keine Hashtag-Wüsten (max 2 pro Post)

## Thread-Formel (5-8 Tweets)

```
Tweet 1: HOOK — schockierende Aussage oder Zahl
Tweet 2: Problem Setup — was ist kaputt/falsch?
Tweet 3: Wendepunkt — was hast du gelernt/entdeckt?
Tweet 4-6: Fleisch — die eigentliche Methode/Lösung
Tweet 7: Beweis — Ergebnis, Zahl, Screenshot
Tweet 8: CTA — Follow / Link / Antwort-Aufforderung
```

## Lead-Generierung

### Wer ist ein Lead?
- Antwortet auf Posts über BMA, AI Automation, oder Consulting
- Erwähnt "Brandmeldeanlage", "Wartung", "DIN 14675"
- Fragt nach Preis, Tool, Setup
- Hat >500 Follower und engaged regelmäßig

### Lead-Aktion:
1. Antwort auf Thread oder DM
2. Freundlich, direkt, kein Hard-Sell
3. Link zu Gumroad-Produkt oder "Ruf mich an"
4. In CRM eintragen (Port 3500) mit BANT-Score

### DM-Template (kurz):
```
Hey [Name],

hab deinen Post gesehen. [Bezug auf spezifischen Inhalt].
Arbeite an [relevante Lösung] — könnte genau passen.

Kurzes Gespräch? [Calendly oder direkte Antwort]

Maurice
```
